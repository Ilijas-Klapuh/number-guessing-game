# Number Guessing Game

A simple Java number guessing game where the program randomly selects a number and the player tries to guess it. Built using Java in VS Code.

## How to Run

### Prerequisites
- Java JDK installed (Java 17 recommended)
- Optional: VS Code with Java Extension Pack

### Using Command Line
1. Clone the repository
2. Go into the project folder
3. Compile the Java file
4. Run the program
5. Follow the prompts to play.

### Using VS Code
1. Open VS Code and open the project folder.
2. Make sure the Java Extension Pack is installed.
3. Open `game.java`.
4. Click the **Run** button at the top right.
5. Follow the prompts in the terminal to play.
